# Owon_XDM1041 / 2041-Linux-interface
### Supports a simple desktop gui to read and control the DMM. (DC functions only.)</br>
![GUI](DMM1.1.png)



## Usage </br>
First install the requirements
```shell
pip install -r requirements.txt
```

To run the gui
```shell
python3 gui_V11.py
```
